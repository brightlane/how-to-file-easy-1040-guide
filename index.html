🦉 How to File Taxes for Beginners 2026

Form 1040 explained line-by-line • First-time filers only

✅ File Taxes in 4 Steps (15 Minutes)

  1. Jan 31: Get W-2 from employer
  2. Line 1a: Copy W-2 Box 1 → Form 1040
  3. Line 11: Enter $14,600 standard deduction
  4. Line 20: File free → Direct deposit refund

📋 Tax Documents Checklist (Get These FIRST)

  • Form W-2 → Wages + taxes withheld (employer sends by Jan 31)
  • Form 1099-NEC → $600+ side jobs (babysitting, mowing lawns)
  • Form 1099-INT → Bank interest (even 47¢ counts)

📄 Form 1040 Page 1: INCOME LINES

Line 1a: Wages (W-2 Box 1)

  • Copy W-2 Box 1 → Line 1a exactly
  • Summer jobs, retail, restaurant work ALL count
  • Don't include: gifts, allowance, birthday cash

Line 8: TOTAL INCOME

Formula: Line 1a + 1b + 1c + 2b + 3b + 7 = Line 8

Line 20: Refund or Owe?

  • Refund: Line 19 payments > Line 15 tax owed
  • Owe: Line 15 tax > Line 19 payments

Example: $18,000 income - $14,600 deduction = $3,400 taxable → ~$340 tax
W-2 withheld $1,000 → $660 REFUND
